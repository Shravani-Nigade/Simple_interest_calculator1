# Simple Interest Calculator

This project calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## How to Run

1. Run the script:
   python interest.py

2. Enter:
   - Principal amount
   - Rate of interest
   - Time (years)

## Example

Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100# Simple_interest_calculator1
